Javascript Control Flow

JavaScript ternary operator ? :

syntax : condition ? If expression TRUE : If Expression False

let age = 18;
let message ='';
age >= 16 ? (message ="You can Derive"):(message="You can Not Drive");
console.log(message);
//Output : You can Derive.

Multiple JavaScript ternary operators example

let speed = 90;
let message2 = speed >= 120 ? 'To Fast' : speed >= 80 ? 'Fast' : 'OK';
console.log(message2)
// Output: Fast
